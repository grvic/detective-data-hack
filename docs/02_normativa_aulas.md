# Normativa de Uso de Aulas — Instituto Politécnico Valle del Cid

**Versión:** 2.3 | **Actualizada:** Septiembre 2025

## 1. Horario de acceso

- **Horario lectivo:** Lunes a Viernes, 08:00 a 20:00
- **Fuera de horario:** Solo con autorización escrita del jefe de departamento
- **Eventos especiales (hackathons, exámenes):** Se extiende hasta la hora indicada en el acta del evento

## 2. Sistema de control de acceso

- Todas las aulas tienen lector de tarjeta NFC
- Cada acceso queda registrado: tarjeta, ubicación, fecha/hora, tipo (entrada/salida)
- Las tarjetas son personales e intransferibles
- **Importante:** El sistema NO impide la entrada fuera de horario, solo la registra y genera una alerta al día siguiente

## 3. Red WiFi

- Cada aula tiene su propio Access Point (AP-AULA1, AP-AULA2, etc.)
- Los dispositivos se identifican por dirección MAC
- La red queda activa 24 horas para procesos automáticos (CI/CD, backups)
- Se registra: MAC, AP, hora inicio, hora fin, datos transferidos

## 4. Sala de Servidores

- Acceso restringido al personal de IT (Inés Navarro) y dirección
- Contiene los servidores Git internos y el controlador WiFi
- Registro de acceso independiente del sistema general

## 5. Vigilancia nocturna

- Un vigilante realiza rondas a las 22:00, 01:00 y 04:00
- Tiene tarjeta maestra que abre todas las puertas
- Debe reportar cualquier anomalía en el parte diario

## 6. Incumplimientos

- Acceso no autorizado fuera de horario: amonestación
- Uso de tarjeta ajena: expulsión temporal
- Manipulación de equipos ajenos: expediente disciplinario
